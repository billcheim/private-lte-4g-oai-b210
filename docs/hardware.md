# Hardware Requirements

## Core Components

- USRP B210
- Linux host PC or workstation
- USB 3.0 connection for SDR
- LTE-compatible UE device
- Private SIM / USIM cards
- Ethernet / local network access

## Optional Components

- Raspberry Pi management node
- External antennas
- RF attenuators
- GPSDO for synchronization
- Additional UE devices

## Recommended Host Specs

- Multi-core CPU
- 16 GB RAM or higher
- SSD storage
- Ubuntu Linux

## Notes

The USRP B210 is commonly used for lab LTE and SDR experimentation.
