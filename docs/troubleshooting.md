# Troubleshooting

## UE does not detect LTE cell

Check:

- antennas connected
- correct frequency
- correct LTE band
- SDR recognized by host

## UE cannot attach

Check:

- SIM credentials
- PLMN
- TAC
- EPC services running

## No IP address assigned

Check:

- HSS subscriber entry
- SPGW status
- APN configuration

## Poor performance

Check:

- USB 3.0 link
- host CPU load
- RF environment
- gain settings

## Useful Checks

- verify SDR connection
- inspect service logs
- confirm network interfaces
- restart components carefully
