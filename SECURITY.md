# Security Policy

This repository is intended for lab and research environments.

## Recommendations

- Do not expose management interfaces publicly
- Change default credentials
- Restrict SSH access
- Use VPN for remote administration
- Protect SIM credentials and subscriber data
- Review firewall rules before enabling network access

## Reporting Issues

Please report serious security issues privately before public disclosure.

## Scope

Configurations in this repository are examples and are not production-hardened by default.
